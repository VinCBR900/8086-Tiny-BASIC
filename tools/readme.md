##Tools

this folder contains the tiny8086 suite from 

https://github.com/adriancable/8086tiny

But is copied here so CODEX can access it.  We compile with NO_GRAPHICS for STDIO character I/O batch processing

The bootsector is how we test our code by appending the BASIC binary to the bootsector binary as the floppy image - no DOS, just BIOS interrupts
